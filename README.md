# Java 小册

记录一下，我学习 Java Srping Boot 的过程，希望整理出来一个以前端开发者，来看待 Spring Boot 的小册。每个 Demo 都有相应的 Docker Compose 作为环境依赖，这样就算是你不会搭建环境也能一样的学习。

目的是为了快速了解一个领域的知识栈，进行有效的沟通。

因为目前是入门阶段，所以先写 Demo，小册在后续推出。
